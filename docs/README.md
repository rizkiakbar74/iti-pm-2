# Documentation Index

Dokumentasi project sudah dipisah per folder agar root project tetap rapi.

## Folder

- `docs/reports/` - laporan progres dan laporan fase.
- `docs/checklists/` - checklist QA dan deployment.
- `docs/guides/` - panduan instalasi dan menjalankan project.
- `docs/security/` - panduan hardening keamanan.
- `docs/release/` - catatan production/release.
- `docs/testsprite/` - akun, skenario, prompt, dan rencana TestSprite.
- `database/README_DATABASE.md` - panduan database dan schema.

## TestSprite

Untuk test berikutnya, gunakan rencana khusus role matrix:

- `docs/testsprite/08_ROLE_MATRIX_TEST_PLAN.md`

Fokusnya adalah memastikan SUPERADMIN, ADMIN, MODERATOR, dan USER diuji dengan data seed yang jelas, termasuk project, task, assignee, dan notification per role.
