```

## Severity Guide

### Critical
- Login broken for all users.
- Database fatal error.
- User can access forbidden role/admin page.
- User can modify/delete data outside permission.
- Upload allows PHP/script execution.

### High
- Project/task workflow blocked.
- Submit/review cannot complete.
- Notification click broken.
- Role visibility wrong.

### Medium
- UI broken on mobile but workflow still possible.
- Dashboard number inaccurate.
- Activity log missing some context.

### Low
- Text typo.
- Minor spacing/alignment issue.
- Non-blocking style inconsistency.
```
